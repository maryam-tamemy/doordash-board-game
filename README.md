# DoorDash: Scare vs Laugh Touchdown

A 2-player competitive board game built in Java with a full JavaFX GUI,  
developed across 3 milestones for CSEN 401 at the German University in Cairo.

## What It Does
- 100-cell zigzag board inspired by Monsters Inc.
- 8 monster types each with unique passive traits and active powerups
- 9 card types, 5 cell types, energy mechanics, shield/confusion/freeze effects
- Win condition: reach cell 99 with ≥ 1000 energy
- Full JavaFX GUI following MVC architecture

## Tech Stack
Java · JavaFX · OOP · MVC · CSV I/O · Custom Exception Handling

## How to Run
1. Clone the repo
2. Open in IntelliJ IDEA or Eclipse
3. Run `Main.java` inside `game.engine`
4. Choose your role (SCARER or LAUGHER) and play
